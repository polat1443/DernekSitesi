# Dernek Yönetim Sistemi (Web Uygulaması)

Bu depo, sivil toplum kuruluşları ve derneklerin dijital süreçlerini tek bir merkezden yönetmek amacıyla geliştirilmiş kapsamlı bir **Dernek Yönetim Sistemi** projesini içermektedir.

## 📂 Proje İçeriği ve Modüller

| Modül | Açıklama |
| :--- | :--- |
| **Güvenli Kimlik Doğrulama** | E-posta ve şifre doğrulamalı, güvenli giriş (Login) sistemi. |
| **Dinamik Yönetici Paneli** | Veritabanı bağlantılı, tüm site verilerinin kontrol edildiği arayüz. |
| **Üye Yönetimi** | Dernek üyelerinin kayıt, durum ve bilgi takip süreçleri. |
| **İçerik Yönetimi** | Dernek faaliyetlerinin ve duyurularının yönetilmesi. |

## 🛠️ Kullanılan Teknolojiler ve Mimari
* **Backend:** ASP.NET Core (C#)
* **Mimari:** Asenkron Programlama (Async/Await Altyapısı)
* **Veritabanı:** İlişkisel Veritabanı Yönetimi (Database Connection)
* **Güvenlik:** Authentication & Hashing

## 🚀 Gelecek Planları (Roadmap)
Projenin temel mimarisi tamamlanmış olup, aşağıdaki özelliklerin entegrasyonu devam etmektedir:
- [ ] Rol bazlı yetkilendirme (Role-based Authorization)
- [ ] Dosya ve doküman yükleme sistemleri
- [ ] Veri analizi ve raporlama ekranları
